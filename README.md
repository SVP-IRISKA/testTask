# MERN boilerplate
Client/Server boilerplate for pet-projects

## What’s Included?
- React
- Redux
- TailwindCSS
- ESLint
- ExpressJS
- Socket.IO`*`
- Mongoose`*`
- Webpack
`*`*Connected via environment variables.*

## Notice
Minimum supported `Node.js` version is `12.13.0`

## How to use it?
Install modules
```
yarn install
```

Run a project in development mode
```
yarn run dev
```

Run server only
```
yarn run watch:server
```

Run client only
```
yarn run watch:client
```

Build a project
```
yarn run build
```
